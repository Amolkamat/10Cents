# 10Cents

A React Redux App !

### Getting Started

There are two methods for getting started with this repo.

#### Familiar with Git?
Checkout this repo, install dependencies, then start the gulp process with the following:

```
> git clone https://github.com/ilamarang/10Cents.git
> cd 10Cents
> npm install
> npm start
```

```
> npm install
> npm start
```
